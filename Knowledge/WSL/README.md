# WSL & Linux Basics

Welcome! This mini-course teaches you **WSL (Windows Subsystem for Linux)** and the basic **Linux commands** you need to work comfortably in a terminal — skills you'll use no matter what programming project you go on to work on.

This folder is self-contained and generic — it doesn't assume any particular project. If you're following along as part of a specific project, that project should have its own document explaining any project-specific details; come back here any time you need a refresher on the underlying skills.

## Read these in order

1. **[01 — What is WSL?](01-what-is-wsl.md)** — what WSL is, how to open it, and the single most important idea: how Windows paths and Linux paths refer to the same files.
2. **[02 — Basic Linux Commands](02-linux-commands.md)** — the handful of commands you'll type constantly: `pwd`, `ls`, `cd`, and a few more.
3. **[Exercises](exercises.md)** — practice tasks to check you've really got it, with answers included.

## The one-sentence version

> WSL gives your Windows computer a real Linux command line to work in, and Linux paths like `/mnt/d/...` are just a different way of writing the same Windows folders like `D:\...`.
