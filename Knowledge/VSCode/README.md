# VS Code Basics

Welcome! This mini-course teaches you **Visual Studio Code** (VS Code) — the code editor you'll use to actually read and write code — and how it works together with WSL.

This folder is self-contained and generic. If you haven't done the [WSL mini-course](../WSL/README.md) yet, do that first — file 02 here builds directly on it.

## Read these in order

1. **[01 — VS Code Basics](01-vscode-basics.md)** — the main parts of the VS Code window, and how to open a project.
2. **[02 — VS Code with WSL](02-vscode-with-wsl.md)** — how to make sure VS Code is editing your real Linux files, not a separate Windows copy.
3. **[Exercises](exercises.md)** — practice tasks to check you've really got it.

## The one-sentence version

> VS Code is where you'll read, write, and run code — and when working with WSL, you always want to open it in "WSL mode" so it's editing your actual Linux files.
